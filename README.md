
# Tulips Frontend
## Project Setup

### 1. clone repository
 
### 2. make .env file and copy  .env.example to .env file,



### 3. run 
```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```



### Type-Check, Compile and Minify for Production

```sh
npm run build
```
